이 프로젝트는 IoT/Big data/AI/Cloud 분야가 모여 협업한 융복합 프로젝트입니다.<br>
주제는 빅데이터 기반의 AI 추천 광고 키오스크입니다.<br>
예시로 잡은 타겟 설치 장소는 영등포에 위치한 더 현대 서울 내 층별 키오스크로 프로젝트를 진행하였습니다.<br>
키오스크에 달린 카메라를 이용하여 안면인식을 한 후 성별/나이대를 추출하여 그에 맞는 더 현대 서울 내 입점한 매장의 위치와 관련광고를 띄워주는 키오스크입니다.
